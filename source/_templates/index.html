About ¶

Gridmonger is a free and open-source dungeon mapping tool to replace pen-and-paper when playing oldschool computer role-playing games (cRPGs). It aims to do one thing and do it well:

Be the ultimate keyboard-driven grid-based cRPG mapping companion!

Gridmonger is available for free for Windows and Mac OS X.

Most notable features ¶

  • (Mostly) keyboard-driven modal user interface (similar to Vim and Blender)
  • Create maps as large as 999 levels of varying dimensions, up to 6,666 × 6,666 cells per level
  • Support for region-based maps and all common cRPG map styles
  • Four different navigational/editing modes to optimise map-making while playing the game
  • Handy map manipulation tools (cut/copy/paste, nudge, crop, resize, move/fill/erase area, and many others)
  • Linked teleports, stairways and doors for easy navigation
  • Various cell annotation options (e.g. numbers, icons, comments)
  • Unlimited undo & redo history
  • Extensive theming support & 32 themes included out-of-the-box
  • Text labels
